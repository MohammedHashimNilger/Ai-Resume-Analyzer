# AI Resume Analyzer

An AI-powered web application that analyzes resumes and provides feedback on strengths, weaknesses, and improvements using LLMs.

## Live Demo
Frontend: https://ai-powered-resume-analyzer-omega.vercel.app/


## Features
- Upload PDF resume
- AI-powered resume analysis
- Strengths & weaknesses detection
- Improvement suggestions
- Modern UI with drag-and-drop upload

## Tech Stack
Frontend
- React
- CSS

Backend
- Node.js
- Express
- Groq API

Deployment
- Vercel (Frontend)
- Render (Backend)

## Installation

cd frontend
npm install

Run backend

node server.js

Run frontend

npm start
